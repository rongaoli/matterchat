
# MatterChat code folder

This folder contains the code used for the MatterChat paper.

Folder Structure:
    project_root/
    ├── config/             # Configuration files for running demo code
    ├── data_sample/        # Sample data used for demo and testing
    ├── inference_log/      # Inference logs corresponding to each figure in the paper
    ├── Model/              # Source files for model architecture and inference logic
    ├── model_weight/       # Trained and pretrained model checkpoints
    ├── output_dir/         # Outputs generated by inference scripts
    └── util_com/           # Utility functions used across inference scripts
    └── training_code/           # MatterChat training script along with GCN training script



## Setup Instructions
1. Download all the content of this folder.

2. Create and activate the conda environment:

conda env create -f conda_env.yaml -n test_env
conda activate test_env

2. Scripts Running

Once the environment is set up, run any of the inference scripts:

python inference_***.py

The results will be saved in the `./output_dir/` directory.

## Script Summary

- `inference_cif_fig2.py`: Generates the conversation log output for Figure 2 in the paper.
- `inference_cif_fig4.py`: Generates the conversation log output for Figure 3.
- `inference_vis_fig4.py`: Produces the visualization results for Figure 4.
- `inference_MatterChat.py`: Runs full inference on the test dataset (used for Figure 5 and Table 1, without RAG).
- `inference_MatterChat_RAG_fig5.py`: Runs full inference with retrieval-augmented generation (Figure 5 and Table 1, with RAG).
